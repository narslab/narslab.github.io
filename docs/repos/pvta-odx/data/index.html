Data files used in this project

  1. APC

Contains the boarding stops, routes, trip numbers, and boarding and alighting counts at those stops. The main data is collected using the automatic passenger counter system (APC) installed on the buses.

  1. Bytemark

Contains the usage data collected anonymously from ticketing app users of the network which was presented in 2020. The data collected contains the ticket activation coordinates, the timestamps, fare types and prices, among other data. The data does not contain the stop of the activation however.

  1. GTFS

Contains the detailed bus and route schedules.
